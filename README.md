# NEXUS-7 Air Draw Terminal

A futuristic gesture-controlled air drawing application built with Python, OpenCV, and MediaPipe.

## Features

- Gesture-based air drawing
- Six futuristic drawing modes
- Particle effects
- Lightning effects
- Fire effects
- Neon rainbow brush
- Plasma laser brush
- Glitch ink mode
- Starfield mode
- Real-time hand tracking
- Screenshot capture
- Canvas clearing
- Eraser gesture
- Smooth drawing

## Gestures

| Gesture | Action |
|----------|--------|
| ☝ Index Finger | Draw |
| ✌ Index + Middle | Erase |
| 🤘 Index + Pinky | Next Drawing Mode |
| ✋ Open Hand | Clear Canvas |
| ✊ Fist | Pause Drawing |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| 1-6 | Select Drawing Mode |
| C | Clear Canvas |
| S | Save Screenshot |
| ESC | Exit |

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

## Installation

Clone the repository.

```bash
git clone https://github.com/YOUR_USERNAME/NEXUS-7.git
```

Install dependencies.

```bash
pip install -r requirements.txt
```

Run the application.

```bash
python hand_tracking.py
```

## Requirements

- Python 3.10+
- Webcam
